💡 核心洞察

  • Agent Loop: while tool_use → execute → tool_result 循环是 Agent 的心脏
  • Subagent: fresh context + 只返回摘要 = 父 Agent 保持干净
  • Skill 两层注入: Layer 1 元数据 (~10 tokens) + Layer 2 按需加载，节省 ~90% token
  • Context 压缩: micro (每轮) + auto (超阈值) + manual (LLM 主动)，Transcript 保存保证可追溯
  • 文件任务: 持久化到磁盘，context compression 后仍可恢复；blockedBy 依赖图保证执行顺序
  • 后台任务: threading 并行执行 + notification queue 推送完成通知
  • Teammate vs Subagent: 持久存在 + 双向通信 vs 执行完就销毁 + 单向返回
  • Team Protocols: request_id 关联模式 + 协商式关闭（Teammate 可拒绝）+ Plan 审批
  • Autonomous: idle 轮询 5s + 主动认领任务 + Identity Re-injection（压缩后恢复身份）
